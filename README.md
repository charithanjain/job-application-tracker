# Job Application Tracker

## Live Demo: https://job-application-tracker-client-mnvc.onrender.com/

A web app to track and manage your job applications. Add jobs, update status, set follow-up dates, and monitor progress.

---

## Features

- User signup and login
- Add, update, delete jobs
- Track follow-up dates
- Dashboard with job statuses

---

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Atlas)
- **Authentication:** JWT, bcrypt

---

## Installation & Run

### Backend

```bash
cd backend
npm install
# create .env with PORT, MONGO_URI, JWT_SECRET
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```
